# Networking-Essentials

## Objective

I completed Cisco's Networking Basics course to build a solid understanding of how modern networks work. The course covered everything from network types and communication principles to IP addressing, routing, wireless networking, and application services.

Rather than memorising definitions, the focus was on understanding how devices communicate, how data moves across networks, and how common networking technologies work together in real-world environments.

## Skills Learned

* IPv4 Addressing
* Application Layer Services
* Network Media
* Wireless Access Technologies
* Networking Protocols and Standards
* Network Types
* DHCP Fundamentals
* TCP and UDP Communication
* Routing and Network Segmentation
* Network Troubleshooting Utilities
* ARP Operations
* Gateway Configuration

## Tools Used

* Cisco Networking Academy
* Cisco Packet Tracer Labs
* IPv4 and IPv6 Addressing Exercises
* DHCP Configuration Labs
* Network Testing Utilities
* Wireless Network Simulations
